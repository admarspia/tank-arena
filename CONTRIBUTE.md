# Contributing Guidelines

Thank you for contributing to Tank Arena!  
Follow the rules below to keep the project clean and organized.

---

## How You Can Contribute
You may contribute by:
- Adding new features
- Fixing bugs or glitches
- Cleaning and refactoring code
- Improving documentation
- Adding tests
- Enhancing game logic or performance

---

## Coding Style Rules
To maintain code quality, please follow these guidelines:

- Use clear and meaningful variable and function names.
- Keep indentation consistent (4 spaces).
- Add comments for any logic that is not obvious.
- Avoid duplicated code.
- Keep functions small and focused on one task.
